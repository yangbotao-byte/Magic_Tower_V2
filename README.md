# Magic Tower V2 · 魔塔：星魂纪元

一款使用 **C++、EasyX 和 Visual Studio 2022** 开发的 Windows 横版动作角色扮演游戏。

玩家将在雨夜中获得从天而降的法宝，穿过传送阵进入三座魔塔，击败怪物与首领、收集装备，并完成属于守护者与小菁菁的旅程。

> 当前仓库仅维护 EasyX/C++ 版本。

## 游戏特色

- 中文主菜单、设置、世界故事、角色选择与开场寄语
- 雨夜主世界、闪电与粒子效果、法宝掉落及传送阵
- 三个完整关卡，各关拥有独立地图、普通敌人与 Boss
- 玩家待机、行走、跳跃、普攻和技能逐帧动画
- 敌人追踪、近战攻击、技能、冰冻与受伤反馈
- 法宝跟随、主动技能、冷却时间与紫绿特效
- Boss 战利品掉落、拾取和装备属性加成
- 哥特风角色背包：装备栏、武器栏、法宝栏和角色属性
- 胜利、失败、最终结算与致谢界面
- BGM、音量设置、伤害数字和粒子特效

## 游戏流程

```text
开场寄语
  ↓
主菜单与角色选择
  ↓
雨夜主世界：拾取法宝
  ↓
进入传送阵
  ↓
三章地图选关
  ↓
关卡战斗与 Boss 掉落
  ↓
最终通关、结束语与致谢
```

## 操作方式

| 按键 | 功能 |
| --- | --- |
| `A` / `D` | 左右移动 |
| `J` | 普通攻击 |
| `K` | 跳跃 |
| `L` / `Q` | 角色技能 |
| `E` | 使用已装备法宝 |
| `R` | 拾取法宝或战利品 |
| `1` | 打开或关闭背包 |
| `Enter` | 确认、进入关卡或继续 |
| `U` | 通关后查看终章与致谢 |
| `Esc` | 返回上一级界面 |
| 鼠标 | 菜单选择、地图选关、背包装备操作 |

## 三个关卡

| 章节 | 地图 | 敌人配置 |
| --- | --- | --- |
| 第一关 | 雨夜古堡 | 两名普通敌人 + 最终 Boss |
| 第二关 | 深渊战场 | 普通恶魔 + 超大型 Boss |
| 第三关 | 终焉之塔 | 两名普通敌人 + 最终 Boss |

击败每关最终 Boss 后会掉落对应装备。靠近战利品按 `R` 拾取，物品会进入背包并提供生命、攻击或防御加成。

## 构建方式

### 环境要求

1. Windows 10 或 Windows 11
2. Visual Studio 2022，安装“使用 C++ 的桌面开发”工作负载
3. [EasyX](https://easyx.cn/) 图形库，并安装到对应 Visual Studio 版本

### 编译运行

```bash
git clone https://github.com/yangbotao-byte/Magic_Tower_V2.git
```

1. 使用 Visual Studio 打开根目录下的 `魔塔C++.sln`。
2. 将生成配置选择为 `Debug` 或 `Release`，平台选择 `x64`。
3. 确认项目工作目录为 `$(ProjectDir)`。
4. 按 `F5` 编译并运行。

游戏通过相对路径读取图片和音乐，因此运行时必须保留 `魔塔C++/` 下的资源目录结构。

## 常见问题

### 找不到 `graphics.h`

EasyX 尚未安装或没有集成到当前 Visual Studio。重新运行 EasyX 安装程序，并选择 Visual Studio 2022。

### 能启动但图片加载失败

检查 Visual Studio 项目属性：

```text
配置属性 → 调试 → 工作目录 → $(ProjectDir)
```

不要只复制 `.exe` 文件；图片、动画、音乐目录也必须一起保留。

### 中文显示乱码

项目面向中文 Windows 环境。请在 Visual Studio 中使用与源文件一致的中文编码打开文件，并确保系统安装了微软雅黑等中文字体。

### 音乐没有声音或播放卡顿

检查 `music/` 目录是否完整，并确认系统允许程序使用 Windows 多媒体接口播放 MP3。
